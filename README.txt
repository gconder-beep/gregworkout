GREG'S KETTLEBELL TRACKER

This is a ready-to-host Progressive Web App (PWA).

To install it on a phone, the folder must be hosted over HTTPS. Easy options:
1. GitHub Pages
2. Netlify Drop
3. Cloudflare Pages

After hosting:
- iPhone: open in Safari > Share > Add to Home Screen
- Android: open in Chrome > menu > Install app / Add to Home screen

Workout history is stored locally on the phone/browser. Use Export CSV from the Log tab for backups.
