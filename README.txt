- index.html
- manifest.webmanifest
- sw.js
- assets/

  - illustrations/

GitHub Pages:
Settings > Pages > Deploy from a branch > main > /(root)

After replacing an older version:
1. Wait 1-3 minutes for GitHub Pages to rebuild.
2. Open the site and refresh twice.
3. If the old version remains, clear website data or remove/re-add the Home Screen app.
